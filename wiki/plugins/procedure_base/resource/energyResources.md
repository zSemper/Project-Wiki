# Energy Resources

An energy resource allows the interaction with energy storage from blocks like machines, items or entities.

![Set Block](/resources/plugins/procedure_base/resource/setBlockEnergyResource.png)

![Set Item](/resources/plugins/procedure_base/resource/setItemEnergyResource.png)

![Set Entity](/resources/plugins/procedure_base/resource/setEntityEnergyResource.png)

Each procedure block takes an energy resource variable and gives out a boolean whether the block/item/entity has an energy resource.

---

These procedure blocks return an energy resourec directly, but can cause an error if they are `null`.
