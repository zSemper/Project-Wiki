# Get Frame Number

Gets the number of the frame from a given value, a maximum value and the amount of frames. Useful progress or energy bars in GUIs.

![Get Frame Number](/resources/plugins/procedure_base/misc/getFrameNumber.png)

## Parameter

Get frame number Max frames: `Frames` Current value: `Value` Max Value: `Max`

|Definition|Description|
|-|-|
|Frames|The amount of frames given.|
|Value|The current given value.|
|Max|The maximum the value can be.|
